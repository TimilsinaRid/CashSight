# CashSight

CashSight is a scenario-based financial decision system that helps small businesses forecast cash flow, diagnose risks, and evaluate actions before making decisions.

## Features
- Cash flow forecasting
- Risk detection
- Cause analysis
- Scenario simulation
- Ranked decision recommendations

## Key Insight
CashSight tells you not just that you'll run out of cash, but why — and what to do about it.
